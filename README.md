## Video Website System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/18b28ded181742199569fcd6d1ac04a3/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Video Sharing
5. Video Rankings
6. Communication Forum
7. Message Feedback
8. Personal Center
9. My Posts
10. My Favorites
11. Background Management

**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Video Sharing Management
5. Video Rankings Management
6. Platform Announcement Management
7. Video Type Management
8. Communication Forum
9. Message Board Management
10. System Management (Carousel Management)
